# Diploma
A small molecule generation project for miRNAs combining the RNA-FM and MolGPT models

A graduate project implementing the MolGPT + RNA FM hybrid architecture for conditional generation of molecules (SMILES), taking into account information about RNA sequences. The model is trained on data on the bonds between small molecules and RNA, which makes it possible to generate potential ligands with specified properties.
⚠️ Important: Model weights (.pt) and large datasets are not stored in Git

 Recommended Python 3.9–3.11

Main libraries

3. Download the weights and data

go to Releases.
Download the archive with weights.zip) and unzip it to the weights/ folder.
Download Data_ML.csv and put it in the data/ folder.

# 🚀 Quick start

Checking the RNA FM encoder

python scripts/test_rnafm_forward.py

If the script runs without errors, RNA FM is initialized correctly.

Fusion model training

molgpt/fusion_model.py    

--data_path data/Data_ML.csv 

--weights_dir weights/     

--epochs 50    

--batch_size 32   

--lr 1e-4   

--device cuda

Basic parameters (specify them inside the script if necessary):

--data_path — the path to the CSV with the SMILES, RNA_sequence, target, etc columns.

--weights_dir — where to save checkpoints.

--pretrained_weights — the path to pre-trained MolGPT weights (optional).

--device — cuda or cpu.

The script uses early stopping and saves the best model for the validation loss.

Generation of molecules (inference)

molgpt/fusion_model.py   

--weights weights/trained_model.pt  

--rna_sequence "GGGAAACCC"   

--num_samples 100   

--output gen_results/generated_smiles.csv

What happens:
The checkpoint of the fusion model is loaded.

An RNA sequence is supplied to the input.

The model generates 100 SMILES generated by this sequence.

The result is saved in CSV.

# 🔬 Key features
MolGPT is a GPT—based decoder for generating SMILES strings.

RNA FM is a pre—trained encoder for RNA sequences that provides biologically significant embeddings.

The fusion module combines chemical and biological representations through cross-attention or concatenation.

RDKit + MorganFP — for filtering valid molecules and calculating proximity metrics.
